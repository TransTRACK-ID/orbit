# Nuxt 3 SPA Starter

A minimal Nuxt 3 single-page application scaffold created by [Orbit](https://github.com/Zersya/orbit).

## Getting started

```bash
pnpm install
cp .env .env.local  # optional — .env is created from template variables
pnpm dev
```

## Configuration

| Variable | Description |
|----------|-------------|
| `API_BASE_URL` | Backend API base URL |
| `APP_KEY` | Application encryption key |
